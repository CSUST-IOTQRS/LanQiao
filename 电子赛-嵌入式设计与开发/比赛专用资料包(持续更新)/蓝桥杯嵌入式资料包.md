
# 资料包说明

- 资料包内容比较大，下载比较慢的话就需要打开加速器或者用”魔法”
- 资料包的内容每年会随大赛的进行内容会有些许改动，但是一般改动的内容是现成的驱动代码部分，包含的文档内容只要比赛平台不变就一直是这些

# 资料包的更新

- 仓库中的第十六的的资料包是最新一版的，而且相比第十五届有较大的改动，后续应该不会有太大的变化
